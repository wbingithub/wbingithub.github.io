1.4 process.nextTick(callback)

process.nextTick(callback)的功能是为事件循环设置一项任务，Node.js 会在 下次事件循环调响应时调用 callback。

Node.js 适合 I/O 密集型的应用，而不是计算密集型的应用， 因为一个 Node.js 进程只有一个线程，因此在任何时刻都只有一个事件在执行。如果这个事 件占用大量的 CPU 时间，执行事件循环中的下一个事件就需要等待很久，
因此 Node.js 的一 个编程原则就是尽量缩短每个事件的执行时间。

process.nextTick() 提供了一个这样的 工具，可以把复杂的工作拆散，变成一个个较小的事件。
