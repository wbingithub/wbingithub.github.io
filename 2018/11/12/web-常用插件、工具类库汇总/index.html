前端

前端 常用插件、工具类库汇总

函数库

  • Lodash
  • Underscore
  • Ramda
    • 专门为函数式编程风格而设计，更容易创建函数式 pipeline、且从不改变用户已有数据。
  • outils
    • 日期格式化、url参数转对象、浏览器类型判断、节流函数等常用函数，为避免不同项目多次复制粘贴的麻烦，

动画库

  • Animate.css
    • Animate.css：CSS3 动画库，也是目前最通用的动画库。
  • Anime.js
    • Anime.js：一个强大的、轻量级的用来制作动画的javascript库
  • Hover.css
    • Hover.css：CSS hover 悬停效果，可以应用于链接、按钮、图片等等。
  • wow.js
    • 滚动展示动画，WOW.js 依赖 animate.css，所以它支持 animate.css 多达 60 多种的动画效果。
  • Magic.css
    • css3 animation动画库
  • Waves
    • 点击波纹效果
  • move.js
    • 一个小型的JavaScript库，通过JS来控制一系列的CSS动画顺序执行，使CSS3动画变得非常简单和优雅。

滚动库

  • iscroll
    • 平滑滚动插件
  • BetterScroll
    • iscroll 的优化版，使移动端滑动体验更加流畅
  • mescroll
    • 移动端上拉刷新下拉加载
  • jQuery Scrollbox
    • 图片文字滚动插件
  • liMarquee
    • jQuery无缝滚动插件

轮播图

  • Swiper
    • 常用于移动端网站的内容触摸滑动
  • iSlider
    • 移动端滑动组件
  • slip.js
    • 移动端跟随手指滑动组件，零依赖。
  • OwlCarousel2
    • Owl Carousel 2可以让你创建漂亮的响应式旋转木马的jQuery插件，它支持移动触摸屏，功能十分强大。
  • slick
  • WebSlides
  • jQuery轮播插件slider

滚屏

  • fullpage

弹出框

  • layer
    • 独立维护的三大组件之一（layDate、layer、layim）
  • Bootbox.js
  • dialogBox
    • 基于 jQuery
  • easyDialog

消息通知

  • Notyf
    • 简单的响应式纯js消息通知插件
  • PNotify
    • 页面右上角的提示信息（非弹框提示）
  • overhang.js
    • 是一个JQuery插件显示即时通知、 确认或给定元素中的提示。

下拉框

  • select2

级联选择器

  • ustbhuangyi/picker
    • 移动端最好用的的筛选器组件、联动筛选
  • jQueryDistpicker
    • 移动端最好用的的筛选器组件、联动筛选

颜色选择器

  • Bootstrap Colorpicker 2

时间选择器

  • Moment.js
    • 是一个解析，验证，操作和显示日期和时间的 JavaScript 类库。
  • timeago.js
    • 轻量级的时间转换 Javascript 库

表单验证

  • validator.js
  • jQuery Validation
    • jQuery 表单校验
  • Validform
    • 一行代码搞定整站的表单验证！- Jquery表单验证插件

分页插件

  • pagination
