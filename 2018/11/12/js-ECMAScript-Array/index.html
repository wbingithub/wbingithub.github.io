Array 类型

Array 对象方法

  • 创建数组的基本方式有两种
    • 第一种是使用 Array 构造函数，另外，在使用 Array 构造函数时也可以省略 new 操作符。
    • 第二种基本方式是使用数组字面量表示法
  • 监测数组 instanceof , Array.isArray()
  • 转换方法 toLocaleString() , toString() , valueOf() , join()
  • 栈方法 push(),pop()
  • 队列方法 push() , shift(),unshift()
  • 重排序方法 reverse() , sort()
  • 操作方法 concat() , slice(),splice()
  • 位置方法 indexOf(),lastIndexOf()
  • 迭代方法 every(),filter(),forEach(),map(),some()
  • 归并方法 reduce() , reduceRight()

监测数组

  • instanceof 操作符：存在问题,不止一个全局执行环境,会有各自不同的构造函数。
  • Array.isArray(value) : ECMAScript 5 新增的方法。

转换方法

  • 所有对象都具有 toLocaleString() 、 toString() 和 valueOf() 方法。
  • join() : 默认情况下都会以逗号分隔的字符串的形式返回数组项，可以使用不同的分隔符来构建这个字符串。

栈方法 LIFO（Last-In-First-Out，后进先出）

  • push() ：向数组的末尾添加一个或更多元素，并返回新的长度。
  • pop() ： 用于删除并返回数组的最后一个元素。

队列方法 FIFO（First-In-First-Out，先进先出）

  • push()
  • shift()
  • unshift() : 能在数组前端添加任意个项并返回新数组的长度。

重排序方法

  • reverse() : 反转数组项的顺序
  • sort() : 默认升序排列数组项，以接收一个比较函数作为参数，以便我们指定哪个值位于哪个值的前面；比较函数接收两个参数，如果第一个参数应该位于第二个之前则返回一个负数，如果两个参数相等则返回 0，如果第一个参数应该位于第二个之后则返回一个正数。

操作方法

  • concat()
  • slice() ： 可以接受一或两个参数，即要返回项的起始和结束位置。
  • splice(index,howmany,item1,…..,itemX) ： 主要用途是向数组的中部插入项；还可以对数组项进行删除、插入、替换操作。
    • index ： 必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。
    • howmany ： 必需。要删除的项目数量。如果设置为 0，则不会删除项目。
    • item1, …, itemX ： 可选。向数组添加的新项目。

位置方法 ECMAScript 5 新增

  • indexOf()
  • lastIndexOf()

迭代方法

ECMAScript 5 为数组定义了 5 个迭代方法。

每个方法都接收两个参数：要在每一项上运行的函数和（可选的）运行该函数的作用域对象——影响 this 的值。

传入这些方法中的函数会接收三个参数：数组项的值、该项在数组中的位置和数组对象本身。

根据使用的方法不同，这个函数执行后的返回值可能会也可能不会影响方法的返回值。

  • every() ：对数组中的每一项运行给定函数，如果该函数对每一项都返回 true ，则返回 true 。
  • filter() ：对数组中的每一项运行给定函数，返回该函数会返回 true 的项组成的数组。
  • forEach() ：对数组中的每一项运行给定函数。这个方法没有返回值。
  • map() ：对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。
  • some() ：对数组中的每一项运行给定函数，如果该函数对任一项返回 true ，则返回 true 。

归并方法

ECMAScript 5 还新增了两个归并数组的方法： reduce() 和 reduceRight() 。
这两个方法都会迭代数组的所有项，然后构建一个最终返回的值。
其中， reduce() 方法从数组的第一项开始，逐个遍历到最后。
而 reduceRight() 则从数组的最后一项开始，向前遍历到第一项。

reduce() 和 reduceRight() 的函数接收 4 个参数：前一个值、当前值、项的索引和数组对象。这
个函数返回的任何值都会作为第一个参数自动传给下一项。第一次迭代发生在数组的第二项上，因此第
一个参数是数组的第一项，第二个参数就是数组的第二项。

  • reduce()
  • reduceRight()

使用 reduce() 还是 reduceRight() ，主要取决于要从哪头开始遍历数组。除此之外，它们完全
相同。
