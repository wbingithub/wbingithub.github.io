Vuex

1.1 Vuex 介绍

Vuex 是 Vue 团队提供的 状态管理模式。就是针对 Vue.js 提供的数据驱动解决方案。

Vuex 走的是单向数据绑定模式，我们都知道 Vue 是双向绑定的，这两者还是有点区别的，因为在很多时候单向数据流会更安全、可靠一些。

Vuex 的数据流和核心概念

  • Action
  • State : 用来存储数据，看成一个全局的大json对象.
  • Mutation : 用来修改state,
